## 📄 О репозитории

Этот репозиторий содержит **полный воспроизводимый артефакт** научного исследования:
текст работы, конфигурационные файлы эксперимента, сгенерированный моделями код,
экспертные оценки и итоговые отчёты.

Исследование посвящено сравнительной оценке трёх LLM при генерации кода на языке BSL
(встроенный язык платформы 1С:Предприятие 8) с применением технологии **Model Context Protocol (MCP)**.

Исследуемые модели:
- **Claude Opus 4.5**
- **GPT-5.2 Codex**
- **Gemini 3 Flash**

## 📐 Методика SMOP

Оценка качества сгенерированного кода по 4 критериям:

| Критерий | Описание |
|----------|----------|
| **S** — Syntax | Синтаксическая корректность BSL-кода |
| **M** — Meaning | Семантическая корректность (решает ли код задачу) |
| **O** — Optimization | Оптимальность и качество решения |
| **P** — Platform | Глубина интеграции с метаданными 1С |
